# perses

Perses helm chart

![Version: 0.20.0](https://img.shields.io/badge/Version-0.20.0-informational?style=flat-square) ![Type: application](https://img.shields.io/badge/Type-application-informational?style=flat-square) ![AppVersion: v0.53.0](https://img.shields.io/badge/AppVersion-v0.53.0-informational?style=flat-square)

## Installing the Chart

To install the chart with the release name `my-release`:

```
helm repo add perses https://perses.github.io/helm-charts
helm repo update
helm install my-release perses/perses
```

## Uninstalling the Chart

To uninstall/delete the my-release deployment:

```
helm delete my-release
```
